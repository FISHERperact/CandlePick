# 形态选股展示样本（各前 200 条）

本目录为**与网页展示一致的 200 条样本**，便于仓库克隆后无需本地扫描即可查看效果。

- `品字形_前200条.csv`：品字形形态，前 200 条
- `V字形_前200条.csv`：V 字形形态，前 200 条
- `形态选股结果_展示200条.csv`：合并表（品字形 200 + V 字形 200），首列为「形态」

运行 `scan_viewer` 时，若 `outputs/` 下无全量结果，将自动使用本目录数据展示。

全量结果需在本地执行 `pin_pattern_scan.py` 与 `v_pattern_scan.py` 生成至 `outputs/`。
